# EditorSyntaxMenu

A set of buttons and dropdown menus to apply formats interactively. This helps users who are not familiar with syntactic
writing.

## Import

```ts
import { EditorSyntaxMenu } from 'react-clay-editor';
```

## Props

EditorSyntaxMenu does not take any props.
