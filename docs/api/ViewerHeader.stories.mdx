# ViewerHeader

The component displaying an uneditable header. This is useful to show a fixed title of a document. The header can be
both of a fixed content and a scrollable content.

## Props

| name   | type                                  | requried | default     | description                  |
| ------ | ------------------------------------- | -------- | ----------- | ---------------------------- |
| header | string                                | YES      | -           | text of an uneditable header |
| size   | `'normal'`, `'larger'` or `'largest'` | no       | `'largest'` | font size of the header text |
