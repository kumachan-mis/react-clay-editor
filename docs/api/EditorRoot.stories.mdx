# EditorRoot

The root container of the editor. It receives properties used whole of the components.

## Import

```ts
import { EditorRoot } from 'react-clay-editor';
```

## Props

| name                 | type                                           | requried | description                                        |
| -------------------- | ---------------------------------------------- | -------- | -------------------------------------------------- |
| `text`               | `string`                                       | YES      | syntactic text in the editor                       |
| `setText`            | `Dispatch<SetStateAction<string>>`             | YES      | callback function on the text updated              |
| `syntax`             | `'bracket'` or `'markdown'`                    | no       | syntax of the text                                 |
| `textProps`          | `EditorTextProps`                              | no       | configurations of text                             |
| `itemizationProps`   | `EditorItemizationProps`                       | no       | configurations of itemizations                     |
| `hashtagProps`       | `EditorHashtagProps`                           | no       | configurations of hashtags                         |
| `taggedLinkPropsMap` | `{ [tagName: string]: EditorTaggedLinkProps }` | no       | map of tag names to configurations of tagged links |
| `codeProps`          | `EditorCodeProps`                              | no       | configurations of code strings                     |
| `formulaProps`       | `EditorFormulaProps`                           | no       | configurations of math formulas                    |
| `quotationProps`     | `EditorQuotationProps`                         | no       | configurations of quotations                       |
| `className`          | `string`                                       | no       | CSS className of EditorRoot                        |
| `children`           | `ReactNode`                                    | no       | content of the component                           |

## EditorTextProps

| name                   | type                                                               | default     | description                                                       |
| ---------------------- | ------------------------------------------------------------------ | ----------- | ----------------------------------------------------------------- |
| lineProps              | (lineIndex: number) => <br/>PropsWithoutRef<ComponentProps<'div'>> | `undefined` | function returning props of line elements                         |
| normalLabel            | string                                                             | `'normal'`  | label and the default text of normal sections in the syntax menu  |
| largerLabel            | string                                                             | `'larger'`  | label and the default text of larger sections in the syntax menu  |
| largestLabel           | string                                                             | `'largest'` | label and the default text of largest sections in the syntax menu |
| suggestions            | string[]                                                           | `[]`        | candidates of autocomplete suggestion in headings and plain texts |
| initialSuggestionIndex | number                                                             | `0`         | focused index of autocomplete suggestions when they pop up        |

## EditorItemizationProps
