# ViewerRoot

The root container of the viewer. It receives properties used in the whole of components.

import Api from '../ts/Api';

<div style={{ display: 'flex', justifyContent: 'center' }}>
  <Api colorized="fixed-container" />
</div>

## Import

```ts
import { ViewerRoot } from 'react-clay-editor';
```

## Props

| name               | type                                   | requried | default                                                                                                  | description                                                                                             |
| ------------------ | -------------------------------------- | -------- | -------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------- |
| text               | `string`                               | YES      | -                                                                                                        | syntactic text in the viewer                                                                            |
| syntax             | `'bracket'` or `'markdown'`            | no       | `'bracket'`                                                                                              | syntax of the text                                                                                      |
| palette            | `'light'` or `'dark'`                  | no       | `'light'`                                                                                                | theme of the editor                                                                                     |
| bracketLinkProps   | `ViewerBracketLinkProps`               | no       | see <LinkTo kind="api-editorroot--page#editorbracketlinkprops">EditorBracketLinkProps</LinkTo>           | configurations of bracket links, a subset of EditorBracketLinkProps                                     |
| hashtagProps       | `ViewerHashtagProps`                   | no       | see <LinkTo kind="api-editorroot--page#editorhashtagprops">EditorHashtagProps</LinkTo>                   | configurations of hashtags, a subset of EditorHashtagProps                                              |
| taggedLinkPropsMap | `Record<string, iewerTaggedLinkProps>` | no       | `{}` (see also <LinkTo kind="api-editorroot--page#editortaggedlinkprops">EditorTaggedLinkProps</LinkTo>) | map of tag names to configurations of tagged links, a subset of `Record<string, EditorTaggedLinkProps>` |
| codeProps          | `ViewerCodeProps`                      | no       | see <LinkTo kind="api-editorroot--page#editorcodeprops">EditorCodeProps</LinkTo>                         | configurations of code strings, a subset of EditorCodeProps                                             |
| formulaProps       | `ViewerFormulaProps`                   | no       | see <LinkTo kind="api-editorroot--page#editorformulaprops">EditorFormulaProps</LinkTo>                   | configurations of math formulas, a subset of EditorFormulaProps                                         |
| className          | `string`                               | no       | `undefined`                                                                                              | CSS className of ViewerRoot                                                                             |
| children           | `ReactNode`                            | no       | `undefined`                                                                                              | content of the component                                                                                |

## ViewerBracketLinkProps

`ViewerBracketLinkProps` is a subset of `EditorBracketLinkProps`. The following table shows what properties in
`EditorBracketLinkProps` also exist in `ViewerBracketLinkProps`.

| property in EditorBracketLinkProps | also exists in ViewerBracketLinkProps |
| ---------------------------------- | ------------------------------------- |
| anchorProps                        | YES                                   |
| label                              | no                                    |
| suggestions                        | no                                    |
| initialSuggestionIndex             | no                                    |
| disabled                           | YES                                   |

Refer to <LinkTo kind="api-editorroot--page#editorbracketlinkprops">EditorBracketLinkProps</LinkTo> for details of each
property.
