# Overview

The API of React Clay Editor consists of multiple components. The following examples use all public components.

```tsx
const Editor: React.FC<{ initialText: string }> = ({ initialText }) => {
  const [text, setText] = React.useState(initialText);
  return (
    <EditorRoot text={text} setText={setText}>
      <EditorSyntaxMenu />
      <Divider />
      <EditorTextFieldRoot>
        <EditorHeader header="React Clay Editor" />
        <EditorTextFieldBody />
      </EditorTextFieldRoot>
    </EditorRoot>
  );
};
```

```tsx
const Viewer: React.FC<{ text: string }> = ({ text }) => (
  <ViewerRoot text={text}>
    <ViewerHeader header="React Clay Editor" />
    <Divider />
    <ViewerTextFieldRoot>
      <ViewerTextFieldBody />
    </ViewerTextFieldRoot>
  </ViewerRoot>
);
```

The public components can be split up into four categories: containers of fixed contents, fixed contents, containers of
scrollable contents and scrollable contents.

      scrolled.

  <dt>
    <p>
      EditorTextFieldRoot/ViewerTextFieldRoot (<span style={{ color: colorPalette.scrollableContainer }}>■</span>
      container of scrollable contents)
    </p>
  </dt>
  <dd>
    <p>The containers of scrollable text fields. They grow taller until they fill up the space.</p>
  </dd>
  <dt>
    <p>
      EditorHeader/ViewerHeader (<span style={{ color: colorPalette.fixedContent }}>■</span> fixed content,
      <span style={{ color: colorPalette.scrollableContent }}>■</span>
      scrollable content)
    </p>
  </dt>
  <dd>
    <p>
      The components displaying uneditable headers. They are useful to show a fixed title of a document. They can be
      both of fixed contents and scrollable contents.
    </p>
  </dd>
  <dt>
    <p>
      EditorTextFieldBody/ViewerTextFieldBody (<span style={{ color: colorPalette.scrollableContent }}>■</span>
      scrollable content)
    </p>
  </dt>
  <dd>
    <p>
      The components displaying main texts and controling interactions from a user. Users may scroll them vertically to
      view and edit their text.
    </p>
  </dd>
  <dt>
    <p>
      Divider (<span style={{ color: colorPalette.fixedContent }}>■</span> fixed content,
      <span style={{ color: colorPalette.scrollableContent }}>■</span>
      scrollable content)
    </p>
  </dt>
  <dd>
    <p>A horizontal line to divide contents visually. It can be both of a fixed content and a scrollable content.</p>
  </dd>
