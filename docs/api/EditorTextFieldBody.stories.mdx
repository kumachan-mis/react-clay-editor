# EditorTextFieldBody

The component displaying a main text and controling interactions from a user. Users may scroll it vertically to view and
edit their text.

## Import

```ts
import { EditorTextFieldBody } from 'react-clay-editor';
```

## Props

EditorTextFieldBody does not take any props.
