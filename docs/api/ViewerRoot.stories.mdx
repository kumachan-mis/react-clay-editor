# ViewerRoot

The root container of the viewer. It receives properties used whole of the components.

## Import

```ts
import { ViewerRoot } from 'react-clay-editor';
```

## Props

| name               | type                                         | requried | default                                                                                        | description                                                                                                  |
| ------------------ | -------------------------------------------- | -------- | ---------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------ |
| text               | string                                       | YES      | -                                                                                              | syntactic text in the viewer                                                                                 |
| syntax             | `'bracket'` or `'markdown'`                  | no       | `'bracket'`                                                                                    | syntax of the text                                                                                           |
| textProps          | ViewerTextProps                              | no       | see <LinkTo kind="api-editorroot--page#editortextprops">EditorTextProps</LinkTo>               | configurations of text, a subset of EditorTextProps                                                          |
| bracketLinkProps   | ViewerBracketLinkProps                       | no       | see <LinkTo kind="api-editorroot--page#editorbracketlinkprops">EditorBracketLinkProps</LinkTo> | configurations of bracket links, a subset of EditorBracketLinkProps                                          |
| hashtagProps       | ViewerHashtagProps                           | no       | see <LinkTo kind="api-editorroot--page#editorbracketlinkprops">EditorBracketLinkProps</LinkTo> | configurations of hashtags, a subset of EditorHashtagProps                                                   |
| taggedLinkPropsMap | { [tagName: string]: ViewerTaggedLinkProps } | no       | `{}`                                                                                           | map of tag names to configurations of tagged links, a subset of { [tagName: string]: EditorTaggedLinkProps } |
| codeProps          | ViewerCodeProps                              | no       | see <LinkTo kind="api-editorroot--page#editorcodeprops">EditorCodeProps</LinkTo>               | configurations of code strings, a subset of EditorCodeProps                                                  |
| formulaProps       | ViewerFormulaProps                           | no       | see <LinkTo kind="api-editorroot--page#editorformulaprops">EditorFormulaProps</LinkTo>         | configurations of math formulas, a subset of EditorFormulaProps                                              |
| className          | string                                       | no       | `undefined`                                                                                    | CSS className of ViewerRoot                                                                                  |
| children           | ReactNode                                    | no       | `undefined`                                                                                    | content of the component                                                                                     |

## ViewerTextProps

`ViewerTextProps` is a subset of `EditorTextProps`. The following table shows what properties in `EditorTextProps` also
exist in `ViewerTextProps`.

| property in EditorTextProps | also exists in ViewerTextProps |
| --------------------------- | ------------------------------ |
| lineProps                   | YES                            |
| normalLabel                 | no                             |
| largerLabel                 | no                             |
| largestLabel                | no                             |
| suggestions                 | no                             |
| initialSuggestionIndex      | no                             |

Refer to <LinkTo kind="api-editorroot--page#editortextprops">EditorTextProps</LinkTo> for details of each property.
