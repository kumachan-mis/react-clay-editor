## Fullwidth

`className` is available to overwrite the default styling.

<Canvas>
  <Story name="Fullwidth">
    {() => {
      const [text, setText] = React.useState(
        ['This is a full-width editor.', '`className` is available to overwrite the default styling.'].join('\n')
      );
      return (
        <EditorRoot
          text={text}
          setText={setText}
          className={css`
            && {
              width: 100%;
            }
          `}
        >
          <EditorTextFieldRoot>
            <EditorTextFieldBody />
          </EditorTextFieldRoot>
        </EditorRoot>
      );
    }}
  </Story>
</Canvas>

Note that a className selector in CSS should be **doubled**. Here is an example using `@emotion/css`.

```ts
const className = css`
  && {
    width: 100%;
  }
`;
// two ampersands (&&) is important
```

## With Header

A text in the header is uneditable. It is useful to show a fixed title of a document.
